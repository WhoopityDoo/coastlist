# CoastList — Waterfront Property Zillow Search URLs

These are confirmed coastal waterfront zip codes (FEMA VE flood zone = coastal high hazard).
Click each link to find waterfront homes for sale, then copy individual listing URLs into the CoastList admin panel.

## How to add a property:
1. Click a Zillow search link below
2. Find a property where the lot touches water (check satellite view)
3. Copy the Zillow listing URL (e.g. https://www.zillow.com/homedetails/...)
4. Go to coastlist.com?admin=true
5. Paste the URL, click Fetch, select Water Type, click Publish
